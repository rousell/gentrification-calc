# Gentrification Calculator
This project is intented to map areas of gentrification in Nashville, TN.

## Main Features

1. Median Housing Value, Demographic, and Building Permit Data will be aggregated and displayed in an accessible way via Map API (running leaflet.js and custom tiles)
2. A visitor to the website will be able to add their story about living in Nashville

# Getting Started

1. Install Bower & npm from the assets/lib folder

```
bower install && npm install
```
